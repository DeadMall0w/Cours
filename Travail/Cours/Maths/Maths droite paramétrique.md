# LN
## Formules
$\ln (a \times b) = \ln(a) + \ln (b)$
$\ln(\dfrac{1}{a}) = -\ln a$
$\ln \dfrac{a}{b} = \ln a -\ln b$
$\ln a^{n} = n\times \ln a$
$\ln(\sqrt{ a }) = \dfrac{1}{2} \ln a$
